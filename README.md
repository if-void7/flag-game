# Flag Game

An interactive game where players guess country flags.

## 🚀 Features
- Multiple-choice questions
- Score tracking system
- Instant feedback on answers
- Dynamic interface

## 🛠️ Technologies
- HTML
- CSS
- JAVASCRIPT
- REACT.JS
- VUE

## 🎯 Purpose
This project aims to improve:
- Data handling
- User interaction
- Quiz/game logic development
